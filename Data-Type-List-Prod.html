Documentation

data Prod :: (k -> Type) -> [k] -> Type where Source #

This shows up in many places in the Haskell ecosystem: it's vinyl, or a heterogeneous list parameterized by a functor.

A value of type

Prod f '[a,b,c]

contains an f a, and f b, and an f c.

While this exists in my places, it's reproduced here to to provide a canonical implementation.
